Setup Hackatime

Connect your coding time tracker to enable project submissions

1
Visit Hackatime
First, you need to go to Hackatime and create an account or sign in if you already have one.
Open Hackatime

What is Hackatime?

Hackatime is a time tracking service built for Hack Club that monitors your coding activity across different editors and projects. It's essential for validating project submissions and tracking your coding progress.

2
Sign Up or Sign In
If you don't have a Hackatime account yet, create one using your email. If you already have an account, simply sign in.

New to Hackatime?

Creating an account is free and takes less than a minute. You'll need to verify your email address to get started.

3
Navigate to Settings
Once you're logged in, go to your Settings page to find your API configuration.

Finding Settings

Look for the Settings link in your Hackatime dashboard. It's usually located in the navigation menu or profile dropdown.

4
Find Your API Key
Scroll down on the Settings page until you see the "Config File" section. Your API key will be displayed there.

Important!

Copy the entire API key value after "api_key =" (without the quotes). It should look like a long string of characters and numbers separated by dashes.

5
Enter Your API Key
Paste your Hackatime API key in the field below and click save to connect your account.

Success!

Your Hackatime API key has been saved successfully. You can now submit projects!
